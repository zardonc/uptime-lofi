### 3. 凭证流转闭环 (Mermaid)

```mermaid
graph TD
    Start([用户打开 Dashboard]) --> SilentRefresh[AuthProvider 静默刷新 /api/auth/refresh]

    SilentRefresh --> HasCookie{浏览器有 Refresh Cookie?}
    HasCookie -->|有| RefreshCheck{旧 Token 状态?}
    HasCookie -->|无| LoginGate[LoginGate 渲染登录界面]

    RefreshCheck -->|active| Rotate[双发新 Token + 旧 Token 标记 rotated]
    RefreshCheck -->|rotated| Theft[盗用检测: session_id 全部 revoked]
    RefreshCheck -->|expired| LoginGate
    RefreshCheck -->|无效| LoginGate

    Rotate --> NewCookie[Set-Cookie 覆盖旧 Cookie]
    NewCookie --> Dashboard([Dashboard 主界面])
    Theft --> ErrRevoked[返回 401 + 安全告警]
    ErrRevoked --> LoginGate

    LoginGate --> ModeCheck{UI Lock 是否启用?}
    ModeCheck -->|未启用 默认| InputA[输入 API_SECRET_KEY]
    ModeCheck -->|已启用| InputB[输入 UI_ACCESS_KEY]

    InputA --> RateLimit{登录频率检查}
    InputB --> RateLimit

    RateLimit -->|5次/15min 超限| Err429[返回 429 Too Many Requests retry_after]
    RateLimit -->|未超限| AuthCheck{凭证校验}

    AuthCheck -->|API_SECRET_KEY 匹配| TokenGen[凭证生成]
    AuthCheck -->|UI_ACCESS_KEY Hash 匹配 kv_settings| TokenGen
    AuthCheck -->|EMERGENCY_UNLOCK_KEY 匹配| AuditLog[记录审计日志] --> TokenGen
    AuthCheck -->|校验失败| TrackFail[记录 login_attempts] --> Err401[返回 401 Invalid credentials]

    TokenGen --> GenJWT[签发 JWT: session_id, role=admin, aud, iss, exp=15min]
    TokenGen --> GenRefresh[生成 rawRefreshToken: 双 UUID 拼接]

    GenJWT --> StoreDB[(D1: refresh_tokens 表)]
    GenRefresh --> StoreDB

    StoreDB --> HashToken[存储 token_hash + session_id + status=active + expires_at]
    HashToken --> SetCookie[Set-Cookie: refresh_token; HttpOnly; Secure; SameSite=Strict; Path=/; Max-Age=604800]

    SetCookie --> Response[返回 access_token JSON]
    Response --> Frontend([前端接收凭证])

    Frontend --> MemoryBind[access_token 存入内存变量]
    MemoryBind --> Dashboard

    Dashboard --> DailyUse[日常 API 调用: Bearer + Cookie]
    DailyUse --> TokenExpire{Access Token 过期?}

    TokenExpire -->|否| Continue[正常响应]
    TokenExpire -->|是| Refresh[POST /api/auth/refresh]
    Refresh --> RefreshCheck

    Err401 --> Retry[用户重新输入]
    Err429 --> Wait[等待 retry_after 秒数] --> Retry
    Retry --> LoginGate

    Logout([用户点击登出]) --> LogoutAPI[POST /api/auth/logout]
    LogoutAPI --> Middleware[dashboardAuthMiddleware 验证 JWT + 会话状态]
    Middleware --> Revoke[session_id 所有 refresh_tokens 标记 revoked]
    Revoke --> ClearCookie[Set-Cookie: refresh_token=; Max-Age=0]
    ClearCookie --> ClearMem[清除内存 access_token]
    ClearMem --> LoginGate
```
